# Real Goalkeeper Model Slot

Put a rigged human goalkeeper or football player FBX/GLB/Prefab in this folder.

Required Unity import settings:

- Rig: Humanoid
- Avatar Definition: Create From This Model
- Materials: Extract or remap normally

To replace the current AA robot keeper:

1. Open `Assets/Scenes/BM8PenaltyPrototype.unity`.
2. In Project view, select the imported human goalkeeper FBX/Prefab.
3. Run `BM8 > Replace Goalkeeper With Selected Humanoid`.
4. Press Play.

If the model does not move, the model is not imported as a valid Humanoid rig.
